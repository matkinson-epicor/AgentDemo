# AgentDemo

A .NET 9 ASP.NET Core web application created with Amazon Q Agent.

## Project Overview

This project demonstrates a basic ASP.NET Core MVC application structure with:
- Controllers, Views, and Models following standard ASP.NET Core conventions
- A responsive landing page with application description
- Bootstrap styling for a modern look and feel

## Technologies Used

- .NET 9
- ASP.NET Core MVC
- C#
- HTML/CSS
- Bootstrap

## Project Structure

```
AgentDemo/
├── Controllers/         # Contains MVC controllers
├── Models/              # Contains model classes
├── Views/               # Contains view templates
│   ├── Home/            # Views for the Home controller
│   └── Shared/          # Shared view components
├── wwwroot/             # Static files (CSS, JS, etc.)
├── Program.cs           # Application entry point
└── appsettings.json     # Application configuration
```

## Getting Started

### Prerequisites

- .NET 9 SDK
- Visual Studio 2022 or Visual Studio Code

### Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/AgentDemo.git
   ```

2. Navigate to the project directory:
   ```
   cd AgentDemo
   ```

3. Build the project:
   ```
   dotnet build
   ```

4. Run the application:
   ```
   dotnet run
   ```

5. Open your browser and navigate to:
   ```
   https://localhost:5001
   ```
   or
   ```
   http://localhost:5000
   ```
